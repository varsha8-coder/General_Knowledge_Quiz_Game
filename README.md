# General_Knowledge_Quiz_Game

The General Knowledge Quiz Game is a Python-based interactive program designed to test a user’s knowledge through multiple-choice questions. The program presents a set of questions along with four options for each, and the user must select the correct answer.

The questions are stored in a structured format using a list of dictionaries, where each question includes the question text, answer choices, and the correct option. The program displays each question one by one, accepts the user’s input, and checks whether the answer is correct. Based on the user’s response, the score is updated accordingly.

At the end of the quiz, the program displays the total score and calculates the percentage of correct answers. It also provides feedback such as “Excellent Performance,” “Good Job,” or “Keep Practicing” based on the user’s performance.

This project demonstrates fundamental Python concepts such as loops, conditional statements, lists, dictionaries, and user input handling. It is a beginner-friendly application that helps in understanding how interactive quiz systems work.
